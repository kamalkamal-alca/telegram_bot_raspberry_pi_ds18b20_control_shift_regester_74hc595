# telegram_bot_raspberry_pi_ds18b20_control_shift_regester_74hc595
used telegram for control 4 led shift register, The `shift register` command on a Raspberry Pi receives the following information after sending a request from the Telegram platform (application):  `/temp` ==> Temperature C `/humi` ==> Temperature F `/ledon1` ==> Shift register gpio1 HIGH `/ledoff1` ==> Shift register gpio1 LOW etc...
